Searching for a Run

To look for a specific run you should enter as much as possible details you have in order to find an exact match, or to get a limit of results that matched the
If

  • On RUN level
    • Looking for an especific run: Filling the exact name of the run field
    • Looking for similar RUN match: Clicking on the "Fuzzy Search" checkbox
    • Searching from an especific date: Filling the Date Field
    • Searching for run State: Selecting from the state list Field

    • On Project level
      • Looking for an especific project: Filling the exact name of the run field
      • Looking for similar Project match: Clicking on the "Fuzzy Search" checkbox
      • Searching from an especific date: Filling the Date Field
      • Searching for Project State: Selecting from the state list Field

      • Getting the projects for an especific user: Filling the userid field
